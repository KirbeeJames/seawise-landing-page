Common Issues

Route generation returns an error

If you see "AI could not find a specific sea route", try the following:

1. Check that both origin and destination are valid port names (e.g. "London", "Rotterdam", "Singapore")
2. Try adding the country name (e.g. "Houston, USA" instead of just "Houston")
3. For complex routes, try adding intermediate waypoints using the "+ Add Waypoint" button
4. Use the "Smart Fix" button to correct any overland segments

Fuel calculations seem incorrect

Fuel calculations are based on your vessel's fuel curve (RPM / Speed / Burn Rate). Make sure you have configured accurate values in the Vessel Profile section. The app interpolates between your data points, so adding more entries to your fuel curve improves accuracy.

Data not syncing across devices

Cloud sync requires a registered account. If you're using Guest Access, your data is stored locally in the browser only. Create an account to enable cloud sync across all your devices.

App not loading or showing a blank screen

Try a hard refresh (Ctrl+Shift+R on Windows, Cmd+Shift+R on Mac) to clear the browser cache. If the issue persists, try a different browser or clear your browser data for the site.
